MigratoryData Messaging Demo

This demo uses the MigratoryData Client API for JavaScript to connect to a MigratoryData server, subscribe to one or more topics, and publish messages on specific topics. It displays the real-time messages received for the subscribed topics, as well as various status notifications.

Tip: — Open this page from multiple browsers to connect more subscribers (and publishers) and see how messages are delivered in real-time to all subscribers.
